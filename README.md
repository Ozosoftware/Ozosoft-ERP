# Ozosoft-ERP
Ozosoft ERP - Enterprise Resource Planning includes 20+ modules, 300+ features & Powerful BI. It includes key accounting modules, multi-currency, Banking, Business Partners, Check-books, Checks, Sales, Purchasing, Stock Management, Organization Structure, Item information, Multiple warehouses, Human Resources, Projects Management and Reports. It has modern user interface. Ozosoft ERP is so intuitive almost no learning required.
